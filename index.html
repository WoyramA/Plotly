Belly Button Biodiversity Dashboard

Use the interactive charts below to explore the dataset

This project aims to investigate the microbial species present in the human belly button. By collecting samples from volunteers, researchers were able to identify various species of bacteria. This dashboard allows you to explore the diversity of these bacterial species and compare them across individuals.
